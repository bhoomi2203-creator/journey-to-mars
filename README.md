# journey-to-mars
🚀 Journey to Mars – Interactive Web Experience
🌌 Overview

Journey to Mars is a cinematic and interactive web project that simulates a complete space mission from Earth to Mars. It combines storytelling, animations, and futuristic UI elements to create an immersive experience where users travel through different stages of a Mars mission.

The project is designed to feel like a real spacecraft interface, guiding users through launch, deep space travel, landing, and exploration.

🎯 Features
🚀 Launch Phase
Rocket launch animation from Earth
Countdown sequence (3…2…1… Liftoff)
Real-time system logs (engine ignition, boosters, etc.)
Smooth transition into space
🌌 Space Travel
Starfield / deep space background
Animated spacecraft trajectory (Earth → Mars)
Orbital path visualization
Live telemetry data:
Speed
Distance
Time elapsed
Futuristic HUD (Heads-Up Display)
🔴 Mars Arrival & Landing
Rotating Mars with lighting effects
Atmospheric entry animation
Landing sequence with logs:
Entering atmosphere
Parachute deployment
Landing confirmation
🧭 Mars Exploration
Interactive Mars surface (terrain/map)
Radar scanning animation
Terrain analysis panel
Mission data display:
Temperature
Coordinates
Surface details
💻 Tech Stack
HTML5 – Structure
CSS3 – Styling & animations
JavaScript – Interactivity & logic
